How to Calculate Stair Rise

Divide total rise by your target riser height, round to a whole number of risers, then divide total rise by that number.

Formula: total rise ÷ number of risers = actual riser height

Important Note

Stair code requirements vary by location. Always confirm local code and project requirements before building.

Stair Rise Calculator FAQ

How do I calculate stair riser height?

Divide total rise by the number of risers to get the actual riser height.

Should all stair risers be the same height?

Yes. Riser heights should be consistent for safety and code compliance.

Does this replace local building code?

No. Always check local code requirements before building stairs.
